# shravas-capital-etl-pipeline
# Financial Data ETL Pipeline using Python, GCP, BigQuery, and Cube Cloud  ## Status: Work in Progress 
## Overview  This project implements an ETL pipeline to extract financial metrics data, store it in Google Cloud Storage, and load into BigQuery for analytics and semantic modeling using Cube Cloud.  
## Tech Stack  - Python - Google Cloud Storage - BigQuery - Cloud Functions - SQL - Cube Cloud  
## Features Implemented  - ETL pipeline scripts using Python - Automated data ingestion into BigQuery - Structured analytical tables - Scalable cloud-based architecture 
## Work in Progress  - Semantic layer optimization - Automation improvements - Query performance optimization  
## Architecture  Source → Python ETL → GCS → BigQuery → Cube Cloud → BI Tools
